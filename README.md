# Push Button Controlled LED using Arduino (PICSimLab)

## 📌 Overview
This project demonstrates how to control an LED using a push button in Arduino, simulated using PICSimLab. The LED turns ON when the button is pressed and turns OFF when the button is released.

## 🎯 Objective
- Understand digital input and output in Arduino  
- Learn how to use a push button with INPUT_PULLUP  
- Simulate basic hardware control using PICSimLab  

## 🧰 Components Required
- Arduino Board (PICSimLab)
- LED
- Push Button
- Resistor (220Ω recommended for LED)

## 🔌 Connections
- LED Anode (+)  → Digital Pin D6  
- LED Cathode (–) → GND  
- Push Button one side → Digital Pin D4  
- Push Button other side → GND  

## ⚙️ Working Principle
- The push button is configured using `INPUT_PULLUP`
- Default state (not pressed) = HIGH  
- Pressed state = LOW  
- When button is pressed → LED turns ON  
- When button is released → LED turns OFF
